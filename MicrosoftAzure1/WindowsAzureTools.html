SDK de Microsoft Azure para .NET

Para habilitar la compatibilidad con la compilación de aplicaciones web y servicios Web escalables en Microsoft Azure, instale el SDK de Microsoft Azure para .NET.

El SDK de Microsoft Azure para .NET amplía Visual Studio para habilitar la creación, compilación, depuración, ejecución y empaquetado de aplicaciones web y servicios Web escalables en Microsoft Azure. La instalación incluye el SDK de Microsoft Azure.

Nota: el proyecto que contiene esta página solo proporciona instrucciones para instalar el conjunto de herramientas requerido.
